# GrabrCrawler
A simple Grabr site crawler to return a list of unbidded grabs


### Instalation

- Install node modules
```
npm install
```

- Create your .env configuration with keys got from an authenticated grabr site
```
AUTH_TOKEN ="Token <your_token>"
DEVICE="<your_device>"
```


### Compiling

```
npm run compile
```


### Running
```
npm run run
```
